This is an example file server.

Wait, what?

This server was made with node.js! As a science project (in class), I used node.js with Express to make a HTTP server that could serve static files. This worked out great! It has 404 pages, an index page, and can serve arbitrary files (even in directories!). It can't serve the directories properly, though, so I made a check to see if it's a directory. If it is, 404!
